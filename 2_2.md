2.2 MCenter Server Environment Variable File
==============================================

```
################################# Zookeeper(Required) #################################
# If you are using standalone Zookeeper please provide ip/hostname:port for zookeeper_peers
# If you have zookeeper cluster please provide ip1:port1,ip2:port2...ipn:portn
#######################################################################################

ZK_SERVER_PEERS=<zk ip:port>

################################### MYSQL(Required) ###################################
# This section accepts configuration for mysql server. Please provide:
#### SQL_HOST       : Hostname/IP of SQL server. IP preferred.
#### SQL_PORT       : SQL Server port
#### SQL_USER       : Username of user which has read/write access on mcenterdb
#### SQL_PASSWORD   : Password for user mentioned above
#### SQL_DATABASE   : Name of database, leave default mcenterdb if you don't prefer to change
#######################################################################################

MCENTER_DB_TYPE=mysql
SQL_HOST=<sql.host.ip>/<sql.host.name>
SQL_PORT=<port>
SQL_USER=<user>
SQL_PASSWORD=<password>
SQL_DATABASE=mcenterdb


################################### Instance(Required) #################################
#### MCENTER_SERVER_HOST		    : MCenter server host's public ip or hostname
########################################################################################

MCENTER_SERVER_HOST=<your.host.ip>/<your.host.name>


########################## LDAP (Optional if LDAP is not used) #########################
# This section accepts configuration for LDAP/LDAPS
#### MCENTER_AUTH_MODE                            : Specify ldap if you are using ldap/ldaps otherwise
####                                              leave default
#### MCENTER_AUTH_LDAP_URL                        : ldap url
#### MCENTER_AUTH_LDAP_USER_BASE_DN               : ldap user base distinguished name as shown in
####                                              example below
#### MCENTER_AUTH_LDAP_GROUP_BASE_DN              : ldap group distinguished name user belongs to, as
####                                              shown in example below
#### MCENTER_AUTH_LDAP_USER_CN_OR_UID             : Prefix that needs to be used cn/uid
#### MCENTER_AUTH_LDAP_SECURITY_AUTHENTICATION    : Leave default value simple
########################################################################################

MCENTER_AUTH_MODE=internal
MCENTER_AUTH_LDAP_URL=ldap://localhost:389
MCENTER_AUTH_LDAP_USER_BASE_DN=dc=example,dc=org
MCENTER_AUTH_LDAP_GROUP_BASE_DN=cn=mcenter,dc=example,dc=org
MCENTER_AUTH_LDAP_USER_CN_OR_UID=uid
MCENTER_AUTH_LDAP_SECURITY_AUTHENTICATION=simple


################################## Instance(Optional) ##################################
# This section accepts configuration for MCENTER instance
#### MCENTER_SERVER_PORT              : MCENTER server port to be used.
#### MCENTER_LOG_DIR                  : MCENTER server log directory. Please map this location with host
####                                  using -v option of 'docker run'
#### MCENTER_DATA_RETENTION_PERIOD    : Maximum time MCENTER meta data or logs will be kept.
#### MCENTER_SERVER_JVM_OPTIONS       : Java heap memory size.
# Leave default if you don't want to change or not sure what should be the value
########################################################################################

MCENTER_SERVER_PORT=3456
MCENTER_LOG_DIR=/tmp
MCENTER_DATA_RETENTION_PERIOD=7 days
MCENTER_SERVER_JVM_OPTIONS=""

################################### MCENTER UI(Optional) #################################
# This section accepts port for MCENTER UI
# Leave default if you don't want to change or not sure what should be the value
########################################################################################

MCENTER_UI_PORT=7092
```

You can also copy these env files from MCenter Docker image using the following command (fill in Docker image name):
```
docker create -it --name dummy-mcenter-server <mcenter-server-image> bash && docker cp dummy-mcenter-server:/opt/parallelm/mcenter/conf/mcenter-server-env.list . && docker rm -fv dummy-mcenter-server
```
