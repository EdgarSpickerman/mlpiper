2.3 MCenter Agent Environment Variable File
==============================================

```
# This file contains environment variables for MCenter agent instance
# Please read variable description for more details

################################# Zookeeper(Required) ##################################
# If you are using standalone Zookeeper please provide ip/hostname:port for zookeeper_peers
# If you have zookeeper cluster please provide ip1:port1,ip2:port2...ipn:portn
########################################################################################

ZK_SERVER_PEERS=<zk ip:port>


################################### MYSQL(Required) ####################################
# This section accepts configuration for mysql server. Please provide:
#### SQL_HOST       : Hostname/IP of SQL server. IP preferred.
#### SQL_PORT       : SQL Server port
#### SQL_USER       : Username of user which has read/write access on mcenterdb
#### SQL_PASSWORD   : Password for user mentioned above
#### SQL_DATABASE   : Name of database, leave default mcenterdb if you don't prefer to change
########################################################################################

MCENTER_DB_TYPE=mysql
SQL_HOST=<ip>
SQL_PORT=<port>
SQL_USER=<user>
SQL_PASSWORD=<password>
SQL_DATABASE=mcenterdb


############################# MCENTER Instance(Required) #################################
#### MCENTER_SERVER_HOST  : Hostname/ip address of host where MCENTER server instance is running.
####                      IP preferred.
#### MCENTER_SERVER_PORT  : MCENTER server port to be used. Leave default if you haven't changed while
####                      starting server instance
#### HOST		: Hostname/ip address of host where this MCENTER agent instance is about to
####                      run. IP preferred.
########################################################################################

MCENTER_SERVER_HOST=""
MCENTER_SERVER_PORT=3456
HOST=<your.host.ip>/<your.host.name>


############################ User Details(Required) ##################################
# User to be created inside docker. Processes will be launched as this user
#### HOST_GID       : Host group id
#### HOST_GROUPNAME : Host group name
#### HOST_UID       : Host uid
#### HOST_USERNAME  : Host user name
#### HOME           : User home directory
######################################################################################

HOST_GID=2023
HOST_GROUPNAME=superusergroup
HOST_UID=2023
HOST_USERNAME=<user>
HOME=/tmp


############################################# Instance(Optional) ######################
# This section accepts configuration for MCENTER instance
#### MCENTER_AGENT_PORT               : MCENTER agent port to be used.
#### MCENTER_ENGINE_LOGS_DIR          : Engine(E.g: Spark, Tensorflow user module etc) log directory.
####                                  Please map this location with host using -v option of
####                                  'docker run'
#### MCENTER_LOG_DIR		          : MCENTER agent log directory. Please map this location with host
####                                  using -v option of 'docker run'
#### MCENTER_DATA_RETENTION_PERIOD    : Maximum time MCENTER meta data or logs will be kept.
#### MCENTER_AGENT_JVM_OPTIONS        : Java heap memory size.
# Leave default if you don't want to change or not sure what should be the value
#######################################################################################

MCENTER_AGENT_PORT=4567
MCENTER_ENGINE_LOGS_DIR=/tmp
MCENTER_LOG_DIR=/tmp
MCENTER_DATA_RETENTION_PERIOD=7 days
MCENTER_AGENT_JVM_OPTIONS=""
```

You can also copy these env files from MCenter Docker image using the following command (fill in Docker image name):
```
docker create -it --name dummy-mcenter-agent <mcenter-agent-image> bash && docker cp dummy-mcenter-agent:/opt/parallelm/mcenter/conf/mcenter-agent-env.list . && docker rm -fv dummy-mcenter-agent
```
